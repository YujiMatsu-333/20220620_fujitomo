# 20220620_fujitomo

○操作方法
①'input'フォルダに仕分けたい写真を保存する
②'run.bat'をダブルクリック
③ウィンドウが表示されるため、「画像仕分け開始」ボタンをクリック
④ファイル選択画面が開く。仕分けたい画像を選択
⑤「開く」ボタンをクリック
⑥'output'フォルダに、ファイル名に写っている場所が書かれたファイルが保存される。

※⑥の仕分け作業は少し時間がかかります。
